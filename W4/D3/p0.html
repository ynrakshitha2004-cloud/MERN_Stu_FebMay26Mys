Example: Before clear: username = Rakshuu email = rakshu@gmail.com theme = dark After clear: (empty storage) Session Storage Code Explanation Store Data sessionStorage.setItem("email", "rakshu@gmail.com"); Saves email in session storage. Data exists only while the browser tab is open. Get Data let email = sessionStorage.getItem("email"); console.log(email); Retrieves stored email. Prints it in the console. Remove Data sessionStorage.removeItem("email"); Deletes the email value from session storage.
