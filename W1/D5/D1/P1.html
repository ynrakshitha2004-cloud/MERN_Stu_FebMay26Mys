Projects

Skin Cancer Detection using "CNN & YOLOV8 Algorithms"

This project focuses on developing an intelligent skin cancer detection system using Convolutional Neural Networks (CNN) and the YOLOv8 object detection algorithm. The CNN model is utilized for feature extraction and classification of skin lesion images, while YOLOv8 enhances the system by accurately detecting and localizing suspicious regions in real time. The solution is trained on dermatological image datasets to identify malignant and benign skin lesions with improved accuracy. By integrating deep learning–based image classification and object detection techniques, the project aims to support early diagnosis and assist healthcare professionals in clinical decision-making. This implementation demonstrates strong knowledge in computer vision, deep learning, medical image analysis, and real-time AI systems.

"Stenography Hide Secrete Text Message"

This project focuses on developing a secure communication system using steganography techniques to hide confidential text messages within digital images. The system leverages image-based Least Significant Bit (LSB) encoding to embed secret information without visibly altering the original image. The primary objective is to enhance data privacy by concealing sensitive information in a non-detectable format. The application is built using Python and demonstrates practical implementation of cybersecurity concepts such as data hiding, information security, and digital privacy protection. This project highlights my understanding of secure data transmission and real-world cryptographic techniques.
